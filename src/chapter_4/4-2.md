# 文字列と数値の変換

前述した intParse() に加えて、 toString() も使うことがありますので、覚えておきましょう。

```dart
void main() {
  String numText = '123';
  int number = int.parse(numText); // 文字列を整数に変換
  print('文字列「$numText」を数値に変換: $number');
  
  int value = 456;
  String valueText = value.toString(); // 数値を文字列に変換
  print('数値$valueを文字列に変換: 「$valueText」');
}
```
